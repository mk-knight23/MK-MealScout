# MK MealScout

**Ingredient-Based Recipe & Meal Planner**

Find recipes by ingredients you have. Filter by cuisine, diet, and time.

🔗 **Live**: [https://17-web-culinary-discovery.vercel.app](https://17-web-culinary-discovery.vercel.app)

## Tech Stack

Vue 3.5, TypeScript, Vite 7, Pinia, TheMealDB API

## Getting Started

```bash
npm install
npm run dev
```
